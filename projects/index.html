üîß Tools

Tools for Linux and Windows platforms.

  • AseScale - it automates export of Aseprite files in different scales and formats.
    It currently supports PNG and GIF formats, as well as exports to x1 to x10 scales (ignoring other odds).
    Simply drag-and-drop an Aseprite file and it spits out the exports based on its settings.

  • Model Viewer - a simple viewer for OBJ models in Godot.
    Drag-and-drop an OBJ file to view it on an empty or procedural sky environment.

  • Switchlight - A barebones +/- UI controls for brightnessctl.
    You need brightnessctl installed first in your system: sudo apt-get -y install brightnessctl
    After that it should work just fine. Just use the + and - buttons to adjust your display brightness.
    (I made this as my Fn shortcut keys don't work anymore.)
