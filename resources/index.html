  •   Terminal


  •   wget

      Go to your terminal and type wget -k -p LINK
      Depending on connection speed it should only just take a few seconds or so.

      -k    makes it possible to be browsed offline
      -p    downloads needed files for displaying the page
      LINK    the link of the page you'd like to download

      Examples: Wget Command in Linux with Examples

      If you'd like to download the website as a complete mirror, add the -m flag.
      -m flag creates a complete mirror, a local copy

Art

Applications and extensions for creating art. Mostly are about pixel art.

  • Applications


  • Aseprite
    Probably the most known and popular choice by pixel artists such as me.

    Developed by David Capello and the Aseprite community, it is a powerful sprite editor that can animate
    and export in many supported and known formats. It features groupable layers, tile sets, interactable text,
    extensions via Lua scripting, grouping cels into tags with varying loops, different cel durations, onion skinning,
    brush pressure, custom brushes, blend modes, image sequence via multiple image import, and more.

    It also has an autosave and data recovery feature, as well as a customizable theme.

    It comes in a paid bundle and a trial version. You can also build it from source for free.
    Available to download for Windows, Linux, and macOS platforms.

    Overall, because it is popular for us pixel artists and game developers alike,
    there's countless resources of tutorials and extensions available online which mostly are free.



  • GraphicsGale
    Another popular freeware and a powerful pixel art program for pixel artists, animators and alike.

    Unlike Aseprite, it supports .cur and .ani formats by default. It also features real time editing, i.e.
    edit your sprites while they're being played. Like other pixel programs, it also supports multiple layers,
    onion skinning, sprite sheet export, palette control, and more.



  • Pixelorama
    A nice alternative to Aseprite if you need something versatile and free.
    Animations and tiles are possible. Its seamless tilemap has rectangular, isometric, and hexagonal tiles.
    It also has 3D layer support, CLI automation, metadata to interact with other processes, and extension support.



  • Pixquare
    The best choice for iPad and iPhone artists.

    It offers a collapsible UI to show what's only needed, layers and timeline, tilemap, Aseprite file compatibility,
    timelapse recording, custom brushes, Lospec palettes, and many more.



  • Pixel Studio
    One of the well known pixel art apps for Android. Also available on Steam.

    You can add music to your animations and export them to MP4. Google Drive is available to sync across devices.
    They even have an art platform called Pixel Network to showcase everyone's pieces.



  • dotpict
    A simple pixel art tool for Android.
    It allows GIF creation, offers prompts and challenges, and has a community to share art and get feedback too.

üéÆ Games

  • ü§ñ Game Engines


  • Godot Engine
    A free, open-source game engine. Very lightweight and doesn't have any strings attached, i.e.
    just code right away. It features a 'node-based system' where blocks or Nodes are used
    to build complex and reusable scenes.

    GDScript, its built-in scripting language, is dynamically typed and inspired by Python's simplicity.
    The 4.0 and above releases has support for static typing in order to improve performance.
    There is also a C# release that supports the .NET platform, which also increases performance.

    Godot also has a library of add-ons, as well as templates and guides from its community.
    Has OpenGL support for 3.x releases, and Vulkan support for 4.0 and above.
    While mainly a simple game engine it can build applications too. Users can also animate with it.

    It is released with its permissive MIT license, which allows users to use Godot for any purpose,
    study how it works and modify it, and even distribute both unmodified and changed versions
    (e.g. with other licenses and/or commercially).

    Download: 3.x version | 4.x version
